# Backend

Bootstrap minimo del backend de producto para zCorvus.

## Stack base

- Node.js
- Express
- JSON APIs

## Scripts

- `npm run dev`: arranca el backend con watch nativo
- `npm start`: arranca el servidor
- `npm run check`: valida sintaxis

## Rutas base

- `GET /health`
- `GET /api/v1/ping`

## Nota

Este bootstrap existe para que los agentes puedan empezar a trabajar sobre una base real. La primera feature de producto debe expandir esta estructura, no rehacerla desde cero.
