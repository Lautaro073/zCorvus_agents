# Guias internas

Aqui viven guias operativas del workspace que no son specs de producto, pero que ayudan a ejecutar flujos consistentes.

Ejemplos:

- flujo agente -> GitHub
- convenciones de PR
- procedimientos de rollback o recovery
- guia operativa de agentes para token-context (`token-context-agent-operating-guide-v1.md`)
- guia de despacho autonomo OpenCode (`opencode-autonomous-dispatcher.md`)
- politica de uso Context7 para documentacion externa versionada (`context7-usage-policy.md`)
