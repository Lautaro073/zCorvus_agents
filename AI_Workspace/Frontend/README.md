# Frontend

Bootstrap minimo del frontend de producto para zCorvus.

## Stack base

- Next.js
- React
- TypeScript

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run check`

## Rutas base

- `/`
- `/api/health`

## Nota

Este bootstrap sirve como base real para que los agentes Frontend trabajen sobre una app existente y no sobre una carpeta vacia.
