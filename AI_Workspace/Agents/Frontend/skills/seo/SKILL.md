# seo

## Objective
Improve discoverability and indexing readiness for web pages without compromising UX or maintainability.

## Use when
- Building public-facing pages, landing pages, docs, or marketing routes.
- Updating metadata, headings, canonical links, or structured content.

## Core checks
1. Titles and meta descriptions are unique and descriptive.
2. Heading structure is logical (`h1` then nested sections).
3. Canonical URL is defined when duplicate paths are possible.
4. Images include meaningful `alt` text and optimized delivery.
5. Internal links are crawlable and not JS-only when avoidable.

## Output expectations
- Mention SEO-impacting changes in task/PR summary.
- Keep metadata generation deterministic per route.
